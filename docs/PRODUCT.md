# Product Thinking

The goal is continuous product improvement.

When implementing features consider:

- usability
- clarity
- discoverability
- user flows
- long-term maintainability

---

# Product Insight Mode

You may suggest improvements such as:

- simplifying user flows
- improving onboarding
- better empty states
- improved feedback states
- clearer calls to action

Suggestions should be practical and justified.

---

# Innovation Mode

If a task reveals a better product or UX opportunity, briefly propose it.

Use this format:

Problem  
Proposed improvement  
Why it is better
