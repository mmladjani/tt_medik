# AI Command Cheat Sheet

Use these prompts when working with the coding agent.

---

# Analyze Code

Analyze the relevant parts of the codebase.

Explain:

- current behavior
- architecture decisions
- weaknesses
- improvement opportunities

---

# Propose Solution

Propose the best solution.

Include:

- architecture
- component structure
- data flow
- trade-offs

---

# Implement Solution

Implement the proposed solution.

Follow existing patterns and prefer minimal changes.

---

# Refactor Code

Refactor this code to improve:

- readability
- maintainability
- modularity

Avoid changing behavior.

---

# Debug Issue

Investigate this issue.

Explain:

- root cause
- relevant code paths
- proposed fix

---

# Performance Review

Analyze this code for performance issues and suggest improvements.

---

# UI / UX Analysis

Analyze this page from a UX perspective.

Explain:

- user goals
- friction points
- information hierarchy

Suggest improvements.

---

# Component Architecture

Suggest reusable components for this page or feature.

Focus on composability and reuse.

---

# Self Review

Review the implemented solution.

Check for:

- unnecessary complexity
- performance issues
- architecture weaknesses
- UX improvements
